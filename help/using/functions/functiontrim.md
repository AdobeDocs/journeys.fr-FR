# trim {#trim}

## Catégorie

Chaîne

`trim(<parameters>)`

## Paramètre

| Paramètre | Type |
|-----------|------------------|
| chaîne | chaîne |

## Signature et type renvoyé

`trim(<string>)`

Renvoie une chaîne.

## Exemple

`trim(" Hello ")`

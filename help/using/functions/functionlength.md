# length {#length}

Renvoie le nombre de caractères de l’expression sous forme de chaîne dans le paramètre.

## Catégorie

Chaîne

## Syntaxe de la fonction

`length(<parameters>)`

## Paramètre

* chaîne

## Signature et type renvoyé

`length(<string>)`

Renvoie un entier.

## Exemple

`length("Hello World")`

Renvoie 11.

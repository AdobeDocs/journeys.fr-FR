# round {#round}

Renvoie l’entier le plus proche de l’argument avec des liens arrondis à l’infini positif.

## Catégorie

Maths

## Syntaxe de la fonction

`round(<parameters>)`

## Paramètres

* décimal
* integer

## Signatures et type renvoyé

`round(<decimal>)`

`round(<integer>)`

Renvoie un entier.

## Exemple 

`round(3.14)`

Renvoie 3.

`round(3.54)`

Renvoie 4.

`round(-3.14)`

Renvoie -3.

`round(3)`

Renvoie 3.

# isNotEmpty {#isNotEmpty}

Renvoie « true » si la chaîne du paramètre n’est pas vide.

## Catégorie

Chaîne

## Syntaxe de la fonction

`isNotEmpty(<parameters>)`

## Paramètres

* chaîne

## Signature et type renvoyé

`isNotEmpty(<string>)`

Renvoie une valeur booléenne.

## Exemple

`isNotEmpty("")`

Renvoie false.

`isNotEmpty("hello")`

Renvoie true.
